## Installation
Start Minecraft by 
* enter `minecraft-pi` in the terminal.
* click on **Start Game**
* click on **Create New**
* press `tab` to release focus from Minecraft

In terminal
```
git clone -v https://github.com/Bugnon/minecraft
```
